# Historical Planning Archive

Files in this folder are retained for audit context only. They are not active
implementation plans.

Current work follows `../milestones.md`, `../plan-index.md`, and the task files
under `../tasks/`.

Archived plans may mention retired paths, superseded designs, or files that no
longer exist. Use current task files and `../README.md` for active guidance.

## Indexes

- `plan-index.md`: archived task index.
- `milestones.md`: archived milestone index and archived task-to-milestone map.

## Directories

- `tasks/`: archived task plans.
- `milestones/`: archived milestone plans.

## Archiving A Plan

1. Move the file from `../tasks/` or `../milestones/` into the matching
   directory here.
2. Remove its row from the active index (`../plan-index.md` or
   `../milestones.md`).
3. Add the row to the archived index here.
4. Ids are never reused.
