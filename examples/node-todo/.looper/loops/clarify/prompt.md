# Clarification work cell

A human replied on an issue looper had marked `needs-clarification`. The
recent discussion is included below — the LAST human comment answers (or
declines) the open question.

1. Incorporate the answer: update the acceptance-criteria block, scope block,
   and the durable plan accordingly. Record the resolved question + answer as
   a Decision in the plan.
2. If the answer fully resolves the ambiguity → finish grooming and end your
   summary comment with `looper-verdict: ready`.
3. If it raises a new genuine ambiguity → ask ONE crisp follow-up question and
   end with `looper-verdict: needs-clarification`.

Never re-ask an answered question. Bias to proceeding with stated assumptions.
